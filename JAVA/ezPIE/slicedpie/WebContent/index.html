Welcome to SlicedPie

SlicedPie is a REST service API that provides access to the ezPIE component.

Supported URL and message information will be added as functionality is extended.

Basic call into the REST service should look like the following:

  • URL = http://machine:port/slicedpie/api/v1/data
  • Method = POST
  • Body = { "job": "name of job file" }
  • One job file typically included is "ListDefinitions"
  • Tokens can be added to any job call as additional key-value pairs.
